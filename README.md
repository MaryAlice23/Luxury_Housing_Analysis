# Luxury_Housing_Analysis
A data analytics and Power BI project that analyzes Bangalore’s luxury housing market to uncover trends in bookings, builder performance, buyer preferences, and sales efficiency through interactive dashboards.
 
## Overview
A Business Intelligence and data analytics project that analyzes Bangalore’s luxury housing market to uncover trends in bookings, builder performance, buyer preferences, and sales efficiency. The project uses Power BI to transform raw real estate data into interactive dashboards that support data-driven decision-making.

---

## Key Highlights
- Analyzed quarter-wise booking trends across Bangalore micro-markets  
- Identified top-performing builders based on revenue and ticket size  
- Evaluated booking conversion rates and buyer behavior  
- Assessed the impact of amenities, possession status, and sales channels  
- Built interactive dashboards with KPIs, filters, and drill-through analysis
